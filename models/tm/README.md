# 🤖 여기에 티처블 머신 모델을 넣어주세요

[구글 티처블 머신](https://teachablemachine.withgoogle.com/)에서 학습한 모델을 이 폴더에 넣으면,
앱이 자동으로 인식해서 기본 관상 로직 대신 **나만의 학습 모델**로 MBTI를 판정해요.

## 넣어야 할 파일 (TensorFlow.js 내보내기 3종)

```
models/tm/
├── model.json
├── weights.bin
└── metadata.json
```

## 조건

- 티처블 머신에서 **이미지 프로젝트(표준 모델)** 로 학습할 것
- **클래스 이름은 MBTI 4글자**(예: `ENFP`, `ISTJ`)로 지을 것 — 대소문자는 상관없어요
- 16개 전부가 아니어도 돼요 (예: 4개 유형만 학습해도 동작)

자세한 학습 방법은 프로젝트 루트의 [README](../../README.md#-구글-티처블-머신으로-나만의-모델-만들기)를 참고하세요!
