미설정
토큰 만드는 방법 (1분) ① github.com/settings/personal-access-tokens/new 접속
② Repository access → Only select repositories → face-mbti 선택
③ Permissions → Contents: Read and write, Actions: Read and write 부여 → Generate
④ 토큰을 위에 붙여넣고 저장 — 이 브라우저에만 저장돼요(localStorage). 보기 전용이면 토큰 없이도 됩니다!
